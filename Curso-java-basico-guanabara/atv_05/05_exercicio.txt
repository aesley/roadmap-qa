Aula de exercicios 5

Q.1 Segundo a linguagem de programação Java, assinale a opção incorreta.
a)	O Java diferencia letras maiúsculas de letras minúsculas.
b)	A palavra chave new introduz uma declaração de classe e é imediatamente seguido pelo nome da classe.
c)	O método main é o ponto de partida de cada aplicativo java e deve iniciar com public static void main
(String args[]).
d)	O corpo de cada declaração de classe é delimitado por chaves.
e)	Cada instrução termina com ponto e vírgula.
Resposta: B

Q.2 Uma pilha é uma versão limitada de uma lista encadeada. Novos nós só podem ser adicionados e retirados de uma
pilha pelo topo. Em java a classe Stack possui métodos de manipulação em pilhas. Em qual pacote a classe Stack
está definida?
a)	java.util
b)	java.lang
c)	java.net
d)	java.io
e)	java.swt
Resposta: A

Q.3 Na linguagem de programação JAVA, que palavra chave cria um objeto de uma classe especifica?
a)	MAIN
b)	PUBLIC
c)	CLASS
d)	PRIVATE
e)	NEW
Resposta: E
